## Jekyll, Webpack, Vue

**INSTALL**

```
npm install
gem install jekyll bundler
bundle install
```

**DEV**

```
npm run dev
```

**BUILD**

```
npm run build
```

**BUILD & DEPLOY TO NOW**

```
npm run build-deploy
```
